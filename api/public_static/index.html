How it works

Portable, same-origin, and chat-first

  • The page talks to /v1/messages on the same server.
  • Your token stays on the device unless you choose to remember it.
  • Conversation history is stored locally in that browser.
  • The proxy can still be used by Claude Code CLI at the same time.

If you want another device to use the same session, open the same URL and paste the same token there.
